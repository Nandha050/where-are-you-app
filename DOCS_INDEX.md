## 🎓 Key Concepts

### Glassmorphism
Semi-transparent frosted glass effect with backdrop blur. Creates depth and visual hierarchy.
- **Where**: Bottom sheet, nav dock, floating cards
- **How**: Backdrop blur + translucent background
- **Effect**: Modern, premium appearance

### Neumorphism
Soft, sculptured appearance using subtle shadows and highlights. Creates tactile feel.
- **Where**: Cards, buttons, indicators
- **How**: Layered shadows, soft colors
- **Effect**: Soft, approachable design

### Time-First Data Model
Backend sends time data (arrivals, departures) not distance-based calculations.
- **Why**: More accurate, natural for transit UI
- **Format**: ISO time strings (12h format display)
- **Impact**: Cleaner, simpler frontend code

### Gesture-Driven Interactions
Primary interactions via swipe, tap, and drag gestures.
- **Swipe up**: Expand bottom sheet
- **Swipe down**: Collapse bottom sheet
- **Tap nav**: Navigate to section
- **All responsive**: <100ms latency
